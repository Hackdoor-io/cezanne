Cezanne is a simple Go program built to run on AWS lambda. <br />
It generates images from web pages using [ChromeDP](https://github.com/chromedp).

# Building

```bash
# Install dependencies
make deps

# Build
make build

# Build and run program
make run

# Build lambda function
make lambda
```
